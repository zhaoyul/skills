# epub-to-vibevoice

Local skill for converting EPUB books into clean, narration-friendly text for VibeVoice audiobook generation.

## Structure

- `SKILL.md` — skill entrypoint
- `examples/basic-workflow.md` — minimal end-to-end workflow
- `references/tuning-notes.md` — chunking and stability notes
- `references/troubleshooting.md` — common problems and fixes
- `resources/` — legacy paths kept for backward compatibility
